# SNUDAAE-LATEX-THESIS
서울대학교 건축학과  박사학위논문 템플릿

1) thesis\config.tex 에서 기본정보 수정
2) thesis\figure 폴더에 필요한 그림 넣고 (*.pptx는 gitignore설정되어있으므로 별도관리 또는 gitignore해제)
3) thesis\reference.bib 파일에 필요한 ref 넣고
4) thesis\body.tex에서 1장부터 n장까지 모두 작성 (이 외 파일 수정할 필요 없음)
5) table은 \newfullwidthtable, \newcontenttable, \newlongtable 활용
